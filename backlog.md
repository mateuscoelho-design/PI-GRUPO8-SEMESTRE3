# Backlog Inicial do Projeto
Aplicativo Móvel para Monitoramento de Máquina de Corte

---

## Visão Geral

Este backlog apresenta os épicos iniciais do projeto de desenvolvimento móvel responsável por capturar dados de uma máquina de corte, realizar a contagem de produção e fornecer visualizações, relatórios e resumos inteligentes com apoio de IA.

---

# Épicos do Projeto

---

## ÉPICO 1 – Gerenciamento de Usuários

**Objetivo:** Permitir controle de acesso e gerenciamento de perfis dentro do aplicativo.

### Funcionalidades:
- Cadastro de usuário
- Login e autenticação
- Recuperação de senha
- Edição de perfil

---

## ÉPICO 2 – Gerenciamento de Máquinas (Conexões)

**Objetivo:** Permitir cadastro e gerenciamento das máquinas de corte conectadas ao sistema.

### Funcionalidades:
- Cadastro de nova máquina
- Edição de dados da máquina
- Remoção de máquina
- Teste de conexão
- Configuração de metas de produção
- Associação de máquina a usuário

---

## ÉPICO 3 – Visualização de Dados e Dashboard

**Objetivo:** Permitir análise visual da produção por meio de gráficos e indicadores.

### Funcionalidades:
- Dashboard inicial com resumo geral
- Gráfico de produção diária
- Gráfico semanal e mensal
- Filtro por período
- Filtro por máquina
- Indicador de meta atingida
- Comparação entre períodos

---

## ÉPICO 4 – Relatórios e Resumos com IA

**Objetivo:** Gerar análises automáticas e insights inteligentes a partir dos dados coletados.

### Funcionalidades:
- Resumo automático da produção diária
- Comparação com períodos anteriores
- Identificação de padrões de produtividade
- Alertas de queda de desempenho
- Sugestões baseadas em dados históricos
