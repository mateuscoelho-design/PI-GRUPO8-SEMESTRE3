## Requisitos Não Funcionais (RNF)

### RNF01 — Tempo de resposta
O sistema deve carregar telas principais em até **2 segundos**.

### RNF02 — Segurança
As senhas devem ser armazenadas usando hash seguro.

### RNF03 — Compatibilidade
O sistema deve funcionar nos sistemas ANDROID e IOS.

### RNF04 — Formatação
O sistema deve manter a formatação igual em todos os dispositivos móveis.
