# Gráfico de Produção Diária

## Descrição
Exibe a produção diária das máquinas em formato de gráfico.

---

## User Story

Como usuário do sistema  
Quero visualizar gráficos diários de produção  
Para acompanhar o desempenho diário.

---

## Caso de Uso

**Nome:** Visualizar Gráfico Diário  
**Ator:** Usuário

### Fluxo Principal

1. O usuário acessa o dashboard.
2. O sistema gera gráfico com dados de produção diária.
3. O gráfico é exibido ao usuário.

---

## Critérios de Aceitação

- O gráfico deve apresentar produção por dia.
- Os dados devem vir das máquinas cadastradas.
