# Recuperação de Senha

## Descrição
Permite que o usuário recupere o acesso à conta caso esqueça a senha.

---

## User Story

Como usuário do sistema  
Quero recuperar minha senha  
Para voltar a acessar minha conta.

---

## Caso de Uso

**Nome:** Recuperar Senha  
**Ator:** Usuário

### Fluxo Principal

1. O usuário acessa a opção "Esqueci minha senha".
2. O sistema solicita o e-mail cadastrado.
3. O sistema envia um link de redefinição de senha.
4. O usuário redefine sua senha.

---

## Critérios de Aceitação

- O sistema deve enviar link de redefinição por e-mail.
- O link deve expirar após determinado tempo.
