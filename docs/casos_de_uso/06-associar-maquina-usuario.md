# Associação de Máquina a Usuário

## Descrição
Permite associar uma máquina a um operador responsável.

---

## User Story

Como administrador  
Quero associar uma máquina a um operador  
Para organizar a responsabilidade de uso.

---

## Caso de Uso

**Nome:** Associar Máquina a Usuário  
**Ator:** Administrador

### Fluxo Principal

1. O administrador seleciona uma máquina.
2. O administrador seleciona um usuário.
3. O sistema registra a associação.

---

## Critérios de Aceitação

- O sistema deve permitir selecionar usuário e máquina.
- A associação deve ser salva.
