## Regras de Negócio (RN)

### RN01 — Maioridade
Para criar uma conta, o usuário deve ter mais de 18 anos.

### RN02 — Inatividade
Usuários inativos por mais de 90 dias devem logar novamente no sistema.

### RN03 — Limite de tentativas de login
Após 5 tentativas inválidas, a conta deve ser temporariamente bloqueada.

### RN04 — Apenas usuários da empresa podem criar uma conta
Por meio de validação de domínio de e-mail, filtrar quem pode criar uma conta no sistema.
